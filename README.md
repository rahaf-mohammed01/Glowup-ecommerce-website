# GlowUp – E-commerce Website

## Overview
GlowUp is a full e-commerce website that allows users to browse products and explore an online shopping experience through a clean and user-friendly interface.

The project was developed to practice building dynamic web applications using PHP for backend logic and modern frontend technologies for the user interface.

## Features
- Browse products
- View product details
- Dynamic content using PHP
- Responsive user interface
- Interactive elements using JavaScript
- Organized product display

## Technologies Used
- PHP
- HTML
- CSS
- JavaScript

## How It Works
The website uses PHP to handle backend logic and dynamically display content. The frontend is built using HTML, CSS, and JavaScript to create an interactive and responsive user experience.

Users can browse products and view their details, while the system manages and displays data through server-side processing.
